# Secure Collaboration: The GitHub Connector

Your code is your most valuable asset. Codiner's **GitHub Connector** provides a secure, streamlined way to link your development environment with the world's most popular version control platform.

## Why Connect to GitHub?

Connecting your project to GitHub unlocks a world of collaboration and safety:
- **Cloud Backup**: Every line of code is backed up in the cloud.
- **Version History**: Easily roll back to any previous point in time if something breaks.
- **Collaboration**: Invite teammates to contribute to your repository.

## The Codiner Device Flow

We use GitHub's secure **Device Flow** for authentication. This means:
1. You never have to share your GitHub password with Codiner.
2. You authorize the application through GitHub's official website.
3. Your access tokens are stored securely in your system's encrypted storage (SafeStorage).

## Managing Your Repositories

From within Codiner, you can:
- Create new repositories for your local apps.
- Push your latest changes with a single click.
- Pull updates from your team.
- Manage branches without ever leaving the IDE.

By removing the complexity of CLI Git commands, Codiner makes professional-grade version control accessible to everyone.
